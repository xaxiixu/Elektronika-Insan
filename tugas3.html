Tugas 3: eksperimen menggunakan aplikasi proteus

penjelasan:

Proteus adalah sebuah perangkat lunak yang membantu Anda merancang dan memeriksa PCB (Printed Circuit Board) sebelum mencetaknya. Dengan fitur simulasi seperti PSPICE, Anda bisa melihat bagaimana rangkaian skematik bekerja sebelum dijadikan PCB. Proteus terdiri dari dua program utama: ISIS untuk membuat skematik rangkaian dan ARES untuk merancang layout PCB. Ini sangat berguna untuk desain mikrokontroler dan belajar elektronika, karena menyediakan contoh aplikasi desain yang bisa dipelajari.

Berikut merupakan sourcecode untuk menampilkan “Hello World!” pada LCD.

// include the library code:
#include < LiquidCrystal.h >

// initialize the library by associating any needed LCD interface pin
// with the arduino pin number it is connected to
const int rs = 12, en = 11, d4 = 5, d5 = 4, d6 = 3, d7 = 2;
LiquidCrystal lcd(rs, en, d4, d5, d6, d7);

// make some custom characters:
byte heart[8] = {
0b00000, 0b01010,
0b11111, 0b11111,
0b11111, 0b01110,
0b00100, 0b00000 };

byte smiley[8] = {
0b00000, 0b00000,
0b01010, 0b00000,
0b00000, 0b10001,
0b01110, 0b00000 };

byte frownie[8] = {
0b00000, 0b00000,
0b01010, 0b00000,
0b00000, 0b00000,
0b01110, 0b10001 };

byte armsDown[8] = {
0b00100, 0b01010,
0b00100, 0b00100,
0b01110, 0b10101,
0b00100, 0b01010 };

byte armsUp[8] = {
0b00100, 0b01010,
0b00100, 0b10101,
0b01110, 0b00100,
0b00100, 0b01010 };

void setup() {
// initialize LCD and set up the number of columns and rows:
lcd.begin(16, 2);

// set the cursor to the top left
lcd.setCursor(0, 0);

// Print a message to the lcd.
lcd.print("Hello ");
lcd.print(" World! "); }
void loop() {
// read the potentiometer on A0:
int sensorReading = analogRead(A0);
// map the result to 200 - 1000:
int delayTime = map(sensorReading, 0, 1023, 200, 1000);
// set the cursor to the bottom row, 5th position:
lcd.setCursor(4, 1);
// draw the little man, arms down:
lcd.write(3);
delay(delayTime);
lcd.setCursor(4, 1);
// draw him arms up:
lcd.write(4);
delay(delayTime); }
